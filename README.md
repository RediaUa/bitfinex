# Order Book App

This repository contains a sample Order Book App built using Expo.
## Getting Started
Clone the Repository
```bash
git clone git@github.com:RediaUa/bitfinex.git
```
Install Dependencies
```bash
cd bitfinex
npm install
```
Start the app using Expo:
```bash
npm run start
```
